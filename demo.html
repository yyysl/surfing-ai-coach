使用说明

1

配置OpenAI API Key

点击右下角的设置按钮，输入您的OpenAI API Key以启用AI分析功能

2

上传冲浪视频

支持MP4、MOV、AVI格式，最大200MB，拖拽或点击上传

3

开始AI分析

点击"开始AI分析"按钮，系统将自动识别冲浪者、分析动作并生成标注

4

查看分析结果

观看带标注的分析视频，查看技术评分和详细的动作建议
